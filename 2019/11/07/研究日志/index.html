下午

本来有Adrian的讨论班，但是临时取消了。与几个博士生讨论一下深度学习（Deep Learning）的问题，了解了几个基本问题。

读M统计量的文章。今列举如下：

  1. Matsuda (1981): 1-dim case
  2. Pfanzagl (1973): 多维，但是考虑的是一个Asymptotic minimum contract estimators, 虽然速度可以最优，但条件太强。
  3. Paulauskas (1996): 速度是 \(\log n / \sqrt{n}\), 对于Huber estimator: 只有 \(n^{-1/4} \log n\).
  4. Benkus, Blozenelis and Gotze (1997):

    • --一维-- 情形

    • 假设了：

      \[ \begin{gathered} \mathbb{P} [ Q_N(t) \text{ is convex on } (t_0 - \delta, t_0 + \delta) ] \geq 1 - B N^{-1/2}, \\ \mathbb P [ | t_N - t_0 | \geq \delta ] \leq A N^{-1/2}, \end{gathered} \]

    • 结果：

      \[ d_K \leq C N^{-1/2} \quad \text{provided that } N / \log N \geq \text{ some constant. } \]
  5. J. Huang's note in early years: M estimators, we can consider some conditions inside.
  6. Niwmiro (1992): M estimator for convex functions.
