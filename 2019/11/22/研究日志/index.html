  • Orlicz norms and the Pisier inequality.

If \(\psi\) is a convex and increasing unbounded function $ : R_+ R_+ $ and \(\psi(0) = 0\), we say \(\psi\) is a Young modulus.

For any random variable \(X\), the \(L_{\psi}\)-Orlicz norm of \(X\) is defined to be \[ \begin{align*} \lVert X\rVert_{\psi} = \inf \biggl\{ c > 0 : E \psi \biggl( \frac{ \lvert X\rvert}{c} \biggr) \leq 1 \biggr\}. \end{align*} \] Specially, the function \(\psi_p = e^{x^p} - 1\) is a Yong modulus for each \(p \geq 1\).

Let \(\psi\) be a Yong modulus of exponential type, then there exists a finite constant \(C_{\psi}\) such that for every sequence of random variables \(\{X_k\}\) \[ \begin{align*} \biggl\lVert \sup_{k} \frac{|X_k|}{\psi^{-1} (k)} \biggr\rVert_{\psi} \leq C_{\psi} \sup_{k} \lVert X_k \rVert_{\psi} \end{align*} \]

If \(\psi\) is a Young modulus of the exponential type and \(\{X_k\}_{k = 1}^m\) is any finite collection of random variables, then \[\begin{align*} \lVert \sup_{1 \leq k \leq m} \lvert X_{k}\rvert\rVert_{\psi} \leq C_{\psi} \psi^{-1}(m) \sup_{k} \lVert X_k\rVert_{\psi}, \end{align*}\] where \(C_{\psi}\) is a finite constant depending only on \(\psi\).

Consider the definition of the covering numbers.

Let \((T, d)\) be a metric space. Let the covering number be \[ N(\epsilon, T, d) = \min \bigl\{k: \text{there exist } t_1, \dots, t_k \in T \text{ such that } T \subset \cup_{i=1}^k B(t_i, \epsilon)\bigr\}. \] Let the packing number be \[ D(\epsilon, T, d) \] be the largest \(k\) for which there exist \(k\) points \(t_1, \dots, t_k\) such that \(d(t_i, t_j) > \epsilon\) if \(i \neq j\).

The metric entropy of \((T, d)\) is \(\log N(\epsilon, T, d)\) and the \(\epsilon\)-capacity is \(\log D(\epsilon, T, d)\).

If \(T \subset R^m\) is totally bounded and \(d\) is the Euclidean distance, then \[\begin{align*} N(\epsilon, T, d) \leq \frac{K_m}{\epsilon^m}. \end{align*}\] Note that \(K_m\) depends on \(m\).

The following theorem is the first result. See Vaart and Wellner (1996), Thm 2.2.4, p. 98.

Let \((T, d)\) be a metric space and let \(\{X(t)\}_{t \in T}\) be a stochastic process indexed by \(T\), and let \(\psi\) be a Young modulus of exponential type such that \[ \lVert X(t) - X(s) \rVert_{\psi} \leq d(s, t), \quad s , t \in T. \] Then there exists a constant \(K\) depending only on \(\psi\) such that for all finite subset \(S \subset T\), \(t_0 \in T\) and \(\delta > 0\), \[ \lVert \max_{t \in S} \lvert X(t)\rvert\rVert_{\psi} \leq \lVert X(t_0)\rVert_{\psi} + K \int_0^D \psi^{-1} (N(\epsilon, T, d)) d \epsilon, \] where \(D\) is the diameter of \((T, d)\). Also, \[ \biggl\lVert \max_{s,t\in S, d(s,t) < \epsilon} \lvert X(t) - X(s) \rvert \biggr\rVert_{\psi} \leq K \int_0^\delta \psi^{-1} (N(\epsilon, T, d)) d \epsilon. \]

Suppose that \(X_1, \dots, X_m\) are random variables such that \[ P (|X_i| > x) \leq 2 \exp \biggl( - \frac{1}{2} \frac{x^2}{d + cx}\biggr), \] for all \(x > 0\) and \(i = 1, \dots, m\) for fixed \(c\) and \(d\). Then there is a universal \(K < \infty\) so that \[ \biggl\lVert \max_{1 \leq i \leq m} \lvert X_i \rvert\biggr\rVert_{\psi_1} \leq K \biggl\{c \log (1 + m) + \sqrt{d} \sqrt{\log (1 + m)}\biggr\}. \]

  • Inequalities for sums of independent processes

The Ottavani Inequality.

Let \(X_1, \dots, X_n\) be independent stochastic processes indexed by an arbitrary set. Then for \(\lambda > 0, \eta > 0\), \[ P^* \biggl(\max_{k \leq n} \lVert S_k \rVert^* > \lambda + \eta \biggr) \leq \frac{P^* ( \lVert S_n\rVert^* > \lambda)}{ 1- \max_{1 \leq k \leq n} P ( \lVert S_n - S_k\rVert > \eta )}. \]
